# OrangeTI-84
OrangeTI-84 was created as a solution to a problem. **Traditional calculators are not powerful enough**. True that now we have full computers and cellular devices, but an omage to where we came from is really cool. That why I designed OrangeTI-84, which has the shell of a Texas Instruments-84, but has the power of modern computing. The "calculator" preserves the original buttons of the TI, but is powered by an OrangePi Zero 3w, and adds two USB-C ports (one for charging and one for data), and an HDMI port for additional display power. The device also was designed to have a 3000mAh LiPo battery.
## Whats the difference?
| TI-84 | OrangeTI-84 |
| ----- | ----------- |
| 15 MHz Zilog Z80 (from 1976) | **14.74 GHz Allwinner A733 (from 2025)** |
| 128 KB RAM, 1.5MB ROM | **4GB RAM** |
| LCD Screen | 2.8in LED Capaciative Touchscreen |
| Datalink between calculators | **Wifi, Bluetooth, and USB-C Connectivity** |
| 4 AAA Batteries | LiPo Battery with Powerboost 1000 protection and USB-C charging port |
| Keypad only | Touchscreen, keypad, and usb/bluetooth keyboard possible |
| Built In Display Only | LED Touchscreen, and DP USB-C |
## What is it used for?
Right now, I am planning to install android, and have it run applications like google, youtube, and instagram. This makes it possible to do infinetly more on this than you ever could on your traditional calculator. Because the way that the calculator is built, it is also possible to install a regular version of linux. I am planning to have the ON button change the display to run a TI-84 Emulator, so that it can appear like a regular calculator if needed. Additionally, I am planning to have the arrow buttons function as a mouse or like the tab+arrows on a pc (have not decided yet...) and you know, have the touchscreen work like a touchscreen.
